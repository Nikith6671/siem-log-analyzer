# Mini SIEM Log Analyzer

A cybersecurity project that collects, analyzes, and visualizes logs with secure authentication.

## Features
- Log generation & storage
- PostgreSQL database integration
- Threat detection (failed login attempts)
- Streamlit dashboard
- Role-based access (admin/user)
- Encrypted passwords (bcrypt)
- Export logs & search functionality

## Tech Stack
- Python
- PostgreSQL
- Streamlit
- Pandas
- bcrypt

## Use Case
Helps monitor system logs and detect suspicious activities like brute-force attacks.

## Security
Passwords are securely hashed using bcrypt.

## Dashboard
Interactive UI with real-time updates, charts, and filters.
